Components / Modal

Modal

Use the modal plugin to layer semantic HTML5 <dialog> windows above the page for confirmations, focused tasks, or fully custom content flows.

The API follows a Bootstrap-like mental model: a trigger declares what it wants to toggle, a target points to the modal instance, and the dialog remains a visual surface you can shape with Tailwind.

Position Values

top-start aligns the panel to the upper-left corner.

top aligns the panel to the upper center.

top-end aligns the panel to the upper-right corner.

start aligns the panel to the middle-left edge.

center aligns the panel to the true center.

end aligns the panel to the middle-right edge.

bottom-start aligns the panel to the lower-left corner.

bottom aligns the panel to the lower center.

bottom-end aligns the panel to the lower-right corner.

Animation Values

fade fades the modal root in and out.

scale fades the root and scales the panel.

slide-down fades the root and drops the panel from above.

slide-up fades the root and lifts the panel from below.

none disables visual motion.

Modal position

Modals can be placed in one of nine positions using the data-tw-position attribute on the dialog root. The panel will be anchored to the specified corner or edge while still centering along the opposite axis.

  • top-start - Anchors the modal to the top-left corner.
  • top - Anchors the modal to the top-center edge.
  • top-end - Anchors the modal to the top-right corner.
  • center - Centers the modal both horizontally and vertically.
  • bottom-start - Anchors the modal to the bottom-left corner.
  • bottom - Anchors the modal to the bottom-center edge.
  • bottom-end - Anchors the modal to the bottom-right corner.

Modal animations

Modals can use one of several preset animations defined by the data-tw-animation attribute on the dialog root. Each mode combines opacity and transform changes on the root and panel to create a cohesive motion effect.

  • fade - Fades the entire modal in and out without movement.
  • scale - Fades the root and scales the panel from 95% to 100%.
  • slide-down - Fades the root and drops the panel from above while fading in.
  • slide-up - Fades the root and lifts the panel from below while fading in.
  • none - Disables visual motion for both root and panel.

Markup

<button
  data-tw-toggle="modal"
  data-tw-target="#modal-center"
>
  Open modal
</button>

<dialog
  id="modal-center"
  data-tw-modal
  data-tw-position="center"
  data-tw-animation="scale"
  data-tw-duration="300"
  class="fixed inset-0 m-0 h-full w-full max-h-none max-w-none border-0 bg-transparent p-6 opacity-0 transition duration-300 backdrop:bg-stone-950/75 backdrop:backdrop-blur-sm"
>
  <section
    data-modal-panel
    class="w-full max-w-lg rounded-3xl bg-white text-stone-950 shadow-2xl transition duration-300"
  >
    ...
  </section>
</dialog>

Notes

The root dialog handles positioning and backdrop behavior.

The panel receives focus automatically when the modal opens.

Preset positions and animations are the recommended public API.

Legacy attributes still work for backwards compatibility, but the preferred contract is entirely data-tw-*-based.
