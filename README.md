# tw.vanilla.js 🚀

> **Tailwind Vanilla JS** — це мій навчальний репозиторій-майстерня, де я крок за кроком прокачую роботу з DOM та створюю семантичні UI-компоненти на чистому JavaScript без важких фреймворків.

Ціль проста: не просто дивитися туторіали, а будувати систему. Менше магії, більше чистої логіки та архітектурного мислення.

## 🧩 Стан компонентів (v0.1+)

Кожен компонент розробляється як ізольований модуль з упором на семантику HTML5, доступність (ARIA) та гнучкість стилізації за допомогою Tailwind CSS.

| Компонент | Статус | Опис |
| :--- | :---: | :--- |
| **Modal** 🪟 | ✅ Готово | Модальні вікна з підтримкою стекінгу та ізоляцією логіки |
| **Dropdowns** 🔑 | ✅ Готово | Випадаючі меню на базі семантичних тегів `<nav>` та `<menu>` |
| **Input Masks** 🔢 | ✅ Готово | Посимвольна маска введення (картки, телефони) з захистом від рекурсії |
| **Carousel** 🎠 | ✅ Готово | Слайдер на ООП-класах з анімацією через CSS-трансформи та автоплеєм |
| **Tabs** 📑 | ✅ Готово | Вкладки з повним розділенням доступу за стандартами W3C ARIA |
| **Accordions** 🪗 | ⏳ В планах | Спойлери та блоки питань/відповідей |
| **Tooltips** 💬 | ⏳ В планах | Спливаючі підказки з динамічним позиціонуванням |

---

## 🛠️ Технологічний стек

* **Language:** Чистий Vanilla JavaScript (ES6+ Classes, Event Delegation)
* **Styles:** Tailwind CSS (Utility-first підхід)
* **Structure:** Семантичний HTML5 + ARIA Roles (W3C Валідація)
